# Plugin: core_system

Plugin hệ thống cho quản lý project, sprint, backlog và truy vấn memory ở mức vận hành.

## Tools
- `upsert_project`
- `create_sprint`
- `update_sprint`
- `create_backlog_item`
- `update_backlog_item`
- `list_project_assets`
- `store_project_memory`
- `search_past_decisions`
- `search_codebase`
- `review_code_architecture`
