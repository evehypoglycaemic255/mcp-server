## 🧰 Main Features

### 🗂️ Semantic Memory
Store notes and context in a way that helps the app find related ideas, not just exact words. This is useful when you want the AI to recall old tasks or related topics.

### 📊 Agile Dashboard
Use the dashboard to see what the server stores. You can review memory, check plugin status, and follow activity in one place.

### 🔌 Dynamic Plugin Sandbox
Test plugins without mixing them into your main work. This helps keep experiments separate from core memory.

### ⚡ Token-Efficient Design
The server keeps prompts lean by sending only the most useful context. This helps reduce repeated text and keeps conversations focused.

### 🧩 MCP Support
The app follows the Model Context Protocol, so compatible AI tools can connect to it with less setup.

## ❓ If Something Does Not Work

If the app does not open or the screen stays blank:
